Base 64

Base 64 is a library that provides abilities to encode and decode data using the syntax presented.

base64.encode(String text)

Encodes your text in base64.

base64.decode(String base64)

Decodes your base64.

FFI library

The FFI library allows calling external C functions and using C data structures from pure Lua code.

local ffi = require("ffi")
ffi.cdef[[
int printf(const char *fmt, ...);
]]
ffi.C.printf("Hello!")

So, let's pick that apart:

① Load the FFI library.

② Add a C declaration for the function. The part inside the double-brackets (in green) is just standard C syntax.

③ Call the named C function — Yes, it's that simple!

Imgui library

Create graphical user interferencs for applications using the imgui library.

imgui.Begin("Name", p_opened)

Begin

Creates the imgui window.

"Name" - Title of the imgui window.

p_opened - Deterimines if the imgui window should be visible.

bool shoulddraw, bool p_opened

Shouldraw - Deterimines if the imgui window should be visible on screen.

p_opened - Returns false when the imngui window is closed otherwise true.

imgui.SetNextWindowPos(100, 50)

SetNextWindowPos

Sets the position of the next imgui window using vector2 coordinates.
